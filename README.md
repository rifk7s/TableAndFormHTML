## 🚀 Tentang
Proyek ini berisi contoh-contoh HTML untuk tujuan pembelajaran, berfokus pada struktur tabel dan implementasi formulir.

## ✨ Fitur
- Implementasi jadwal kuliah menggunakan tabel
- Contoh penggunaan tag tabel HTML
- Formulir login dasar
- Desain responsif

## 📝 Contoh

### Jadwal Kuliah
Jadwal kuliah yang komprehensif diimplementasikan menggunakan tabel HTML dengan kode warna:
- 🟢 Sesi praktikum
- 🔵 Kelas teori
- 💛 Waktu istirahat (ISHOMA)

### Form Login
Implementasi formulir login sederhana dan bersih dengan:
- Kolom username
- Kolom password
- Tombol Submit dan Reset
